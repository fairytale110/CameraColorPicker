# CameraColorPicker
Pick color from camera preview or image

## 摘要

### 本项目主要实现
1、通过手机摄像头捕捉预览界面中的像素位置的颜色RGB值

2、通过拍照或者选取本地图片拾取颜色值

3、通过本地视频拾取颜色值

4、颜色值选取记录

5、社会化分享

### 设计技术知识

1、ConstraintLayout的使用

2、属性动画

3、Camera2预览、拍照、切换相机

4、本地视频播放
